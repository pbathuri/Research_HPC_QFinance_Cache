# Research_MonteCarloSim

This workspace contains the **qhpc_cache** undergraduate research project under the **`jk/`** directory.

## Quick start

```bash
cd jk
python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[data-pipeline]"
pip install -e .
python3 run_demo.py
```

Full documentation, data phase, and tests: see **`jk/README.md`** and **`jk/docs/`**.
